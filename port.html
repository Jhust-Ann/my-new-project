FUNCTIONS OF THE VIEWPORT META ELEMENT
width(device-width): Sets the width of the viewport to the device's width.
initial-scale (1.0): Sets the initial zoom level when the page is first loaded.
minimum-scale: Specifies the minimum zoom level.
maximum-scale: Specifies the maximum zoom level.
viewport-fit: Defines how the viewport should fit the device screen.
"cover" ensures the viewport covers the entire screen without any black bars.
user-scalable: Allows or prevents the user from zooming in
or out of the webpage. "yes" enables zooming, and "no" disables it.
